## Страница регистрации

### URL
[Страница регистрации](https://pyaterochka-team.site/signup)

### Тесты
1. При нажатии на ссылку `Войти` открывается страница входа.
2. При наведении курсора на кнопку `Зарегистрироваться` кнопка меняет свой цвет.
3. При нажатии на любое поле ввода, поясняющий текст (Никнейм, Эл. почта, пароль, подтвердите пароль) становится меньшего размера, перемещается в верхнюю часть поля ввода, и оно(поле) начинает подсвечиваться белым цветом.
4. При нажатии на кнопку `Зарегистрироваться` с пустыми полями, появляются красные рамки вокруг всех полей и информация об обязательном заполнении полей и требованиях к полям.  
5. При попытке регистрации пользователя с никнеймом, состоящим только из цифр, происходит успешная регистрация пользователя.  
**_БАГ_**: регистрация не должна проходить с никнеймом, состоящим только из цифр.  
6. При попытке регистрации пользователя с никнеймом состоящим только из одного символа происходит успешная регистрация пользователя. 
**_БАГ_**: нет ограничений на валидность никнейма.  
7. При вводе в поле email невалидного адреса электронной почты появляется надпись `Email не валиден`.  
8. При вводе логина и валидного адреса электронный почты и при вводе пароля, состоящего меньше чем из 6 символов, ввод подсвечивается красным и появляется надпись `Введите минимум 6 символов`.  
**_БАГ_**: нет ограничений на максимальную длину пароля.  
9. При вводе логина и валидного адреса электронный почты и при вводе разных паролей поле ввода повторного пароля подсвечивается красным и появляется надпись `Пароли не совпадают`.  
10. При попытке регистрации пользователя с существующим адресом электронный почты выводится сообщение `Пользователь с данной почтой уже существует`.  
11. При вводе корректных входных данных происходит переход на страницу профиля.

## Страница поста

### URL
[Страница поста с музыкой](https://pyaterochka-team.site/post/1/1)<br>
[Страница поста с видео и текстом](https://pyaterochka-team.site/post/2/101)

### Тесты

#### Пост текущего пользователя

1. При нажатии на кнопку `Редактирование` происходит переход на страницу редактирования

#### Любой пост

1. При вводе текста в поле для комментариева, в поле ввода появляется введённый текст.
2. При нажатии на кнопку `Отправить`, появляется комментарий с введённым текстом.
**_БАГ_**: Поле ввода должно очищяться после добавления комментария.
3. При нажатии на неактивную иконку лайка, иконка становится активной и число лайков у поста увеличивается на 1.
4. При нажатии на активную иконку лайка, иконка становится неактивной и число лайков у поста уменьшается на 1.

#### Пост с музыкой

1. При нажати на кнопку воспроизведения, начинается воспроизведение трека.
2. При перемещении ползунка трека, трек проматывается на выбранный момент времени.
3. При передвежении ползунка громкости, изменяется громкость проигрываемого трека.
4. При нажатии на кнопку паузы, трек останавливается.
5. При нажатии на иконку неперечёркнутого динамика, икона становится перечёркнутым динамиком, ползунок передвигается в левый край, а громкость проигрывания становится 0.
6. При нажатии на иконку перечёркнутого динамика, икона остаётся перечёркнутым динамиком, ползунок возвращается на позицию до нажатия на иконку неперечёркнутого динамика, а громкость возвращается в соответсвующее значение.
**_БАГ_**: Иконка должна превращаться в неперечёркнутый динамик.

#### Пост с видео

1. При нажатии на кнопку проигрывания, начинается воспроизведение видео.
2. При перемещении ползунка видео, видео проматывается на выбранный момент времени.
3. При передвежении ползунка громкости, изменяется громкость воспроизводимого видео.
4. При нажатии на кнопку паузы, видео останавливается.
5. При нажатии на кнопку развернуть видео на полный экран, видео открывается на весь экран.
6. При нажатии на иконку неперечёркнутого динамика, икона становится перечёркнутым динамиком, ползунок передвигается в левый край, а громкость воспроизведения становится 0.
7. При нажатии на иконку перечёркнутого динамика, икона остаётся перечёркнутым динамиком, ползунок возвращается на позицию до нажатия на иконку неперечёркнутого динамика, а громкость возвращается в соответсвующее значение.
**_БАГ_**: Иконка должна превращаться в неперечёркнутый динамик.

## Настройки автора

### URL
[Настройки автора](https://pyaterochka-team.site/profile/edit/creator_settings)

### Тесты

#### Для любого пользователя 

1. При нажатии на надпись `Безопастность`, происходит переход на страницу безопастности.
2. Нажатие на надпись `Основная информация`, происходит переход на страницу с оснавной информацией о профиле.

#### Для пользователя, который уже является автором

1. При нажатии на картинку аватара пользователя, открывается окно загрузки изображения
2. При нажатии на картинку обложки автора, открывается окно загрузки изображения
3. При нажатии на иконку `Добавить уровень`, происходит переход на страницу добавления уровней.
4. При нажатии на кнопку `Редактировать уровень`, происходит переход на страницу редоктирования уровня.

#### Для пользователя, который не является автором

1. При нажатии на поле выбор категории, появляется список доступных категорий.
2. При нажатии на элемент списка с доступными категориями, список пропадёт, выбранная категория появляется в поле ввода.
3. При вводе описание в соответсвующее поле ввода, в поле ввода появляется введённый текст.
4. При нажатии на кнопку `Cоздать автора` без введённой категории, выводится сообщение о том, что не все поля заполенын.
5. При нажатии на кнопку `Cоздать автора` без введённого описания, выводится сообщение о том, что не все поля заполенын.
6. При нажатии на кнопку `Cоздать автора` со всеми заполненными полями, происходит переход на страницу настроек автора для пользователя, который является автором.
